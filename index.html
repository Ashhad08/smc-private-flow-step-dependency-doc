Important Notes

  • ◐ Trims – Conditional Logic:
    • Edit Flow + Same Trim → No reset
    • Different Trim → Vehicle Type, Specs, Color, Price, Additional Info reset
    • Complete Trim Data → Vehicle Type & Specs auto-populated
    • Partial Data → Vehicle Type set, Specs must be selected
  • ★ KmsDriven → Price: Live Ad Edit
    • Same KMs → Price preserved
    • Different KMs → Price fully reset
    • Other modes → Price always reset
  • Additional Information: Cylinders, Seats, Features reset together with Price
  • Same Value Rule: Re-selecting the same value does not cause resets (except Trims in edit mode)
